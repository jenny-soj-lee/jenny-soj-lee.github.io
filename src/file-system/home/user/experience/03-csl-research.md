



## *Undergraduate Research at Zhang Research Group*
## September 2025 - Present

Conduct research under Dr.\ Zhiru Zhang on extending TinyTPU, a compact TPU-inspired accelerator, by exploring new microarchitectures to accelerate machine learning workloads.

Design and verify a vector processing unit in SystemVerilog for neural network inference, implementing bias accumulation, Leaky ReLU activation, and gradient backpropagation, with functional coverage and randomized stimulus using cocotb.

Develop a custom compiler interface that translates Python ML models into accelerator-specific instructions, enabling efficient host–device coordination across the full TPU stack.

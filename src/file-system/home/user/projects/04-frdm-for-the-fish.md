



## *No FRDM for the Fish*

### • Python, C++, Embedded Systems
https://pages.github.coecis.cornell.edu/ece3140-sp2025/sl2854/

Developed an interactive fishing simulation game using the NXP FRDM-KL25Z board's 3-axis accelerometer and push-buttons as primary inputs, interfacing with a multi-page Python Tkinter GUI via UART serial communication for real-time game mechanic control based on board movement.

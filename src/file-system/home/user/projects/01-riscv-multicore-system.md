



## *TinyRV2 Processor and Multicore System*

### • Verilog, PyMTL-3, Python, Computer Architecture

Designed and implemented a fully bypassed, five-stage pipelined TinyRV2 (RISC-V–based) processor in Verilog, including datapath, control logic, and a variable-latency multiplier, then composed the core into complete single-core and quad-core systems.

Built a multicore architecture with four processors, private instruction caches, a shared banked data cache, and a custom ring interconnect, implementing routing, arbitration, and memory-network adapters to support concurrent memory access.

Developed extensive PyMTL3-based verification infrastructure, including directed and randomized tests for pipeline hazards, cache/memory interactions, and multicore execution correctness across the ISA.
